Why These Settings Work for GTX 1660 Super

The GTX 1660 Super features the 6th generation Turing NVENC encoder, which is still capable for 1080p streaming. While not as efficient as newer RTX series, it provides good quality with minimal CPU impact.

The 6GB VRAM is the main limitation - you may need to lower in-game texture settings for newer games while streaming. For pure streaming performance, however, the encoder handles 1080p60 at 6000 Kbps without issues.

Common Issues with GTX 1660 Super Streaming

  • 6GB VRAM: Limited memory means you may need to lower game settings while streaming.
  • Older Encoder: Quality slightly below RTX 30/40 series at the same bitrate.
  • No AV1/HEVC: H.264 only - no codec flexibility.
  • Consider Upgrading: RTX 4060 offers significantly better features at similar power consumption.

Frequently Asked Questions

Is GTX 1660 Super good for streaming?

It's adequate for 1080p streaming but aging. For new streamers, consider the RTX 4060 for better long-term value with AV1 support.

GTX 1660 Super vs RTX 3060 for streaming?

RTX 3060 is significantly better with newer NVENC, 12GB VRAM, and better game performance. Upgrade if budget allows.
