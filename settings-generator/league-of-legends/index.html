Best OBS Settings for League of Legends

Quick Answer: LoL is easy to run, so we can maximize quality. Use 1080p 60fps. Set Lanczos Downscale Filter for sharp text. Bitrate: 6000 Kbps.

Why LoL needs different settings

Unlike shooters, LoL has a static UI and lots of small numbers (gold, damage). If you use "Fast" presets, the text will get blurry during teamfights.

Recommended "Crisp" Settings

Setting Value
Resolution 1080p (Native)
Downscale Filter Lanczos (36 samples)
Encoder Preset P7 (Slowest/Best Quality)

Since LoL is light on GPU usage, you can safely use the P7 (Slow) preset on NVIDIA cards to get the best possible image quality without lagging the game.
