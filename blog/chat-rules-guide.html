Essential Chat Rules

  1. Be respectful - No harassment, hate speech, or discrimination
  2. No spam - Avoid repeated messages, caps lock, or emote walls
  3. English only - (or your preferred language)
  4. No self-promotion - Don't advertise your stream
  5. Listen to mods - Moderator decisions are final
  6. No spoilers - Don't ruin the game for everyone
  7. No backseat gaming - Unless the streamer asks

Where to Display Rules

  • About panel - Twitch panel below stream
  • Chat command - !rules command via bot
  • Stream overlay - On-screen during stream
  • First-time chatter message - Twitch AutoMod feature

FAQ

How do I enforce chat rules?

Use moderators and Twitch's AutoMod. Set up banned words, enable follower-only mode if needed, and empower mods to timeout rule-breakers.

Should I ban or timeout?

Timeout first (300-600 seconds), ban for repeat offenders or severe violations.
