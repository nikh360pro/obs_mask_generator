How to Set Hotkeys in OBS

  1. Open OBS ‚Üí Settings ‚Üí Hotkeys
  2. Find the action you want (e.g., "Start Streaming")
  3. Click the empty box next to it
  4. Press your desired key (e.g., F9)
  5. Click Apply

Pro Tips

  • Use function keys (F1-F12) - Less likely to conflict with games
  • Numpad for scenes - Quick, intuitive switching
  • Avoid common game keys - WASD, E, F, etc.
  • Use Stream Deck - Physical buttons for complex macros

FAQ

Why don't my hotkeys work?

OBS might not have focus. Enable "Focus Workspace" in Settings ‚Üí Advanced, or use a Stream Deck for global hotkeys.

How do I use hotkeys while in a game?

Run OBS as administrator, or use keys that aren't used by the game (function keys work well).
