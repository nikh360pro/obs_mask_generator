Width is always 320 pixels. Height is flexible up to 600px. Keep images under 2.9MB and use PNG for transparency.

Essential Panels Every Streamer Needs

  1. About Me - Who you are, what you stream, streaming schedule
  2. Schedule - Days and times you go live
  3. Rules - Chat rules and expectations
  4. Socials - Links to Twitter, Discord, YouTube
  5. Donate/Support - If you accept donations
  6. PC Specs - Your gaming setup (optional)

Panel Design Tips

  • Keep text readable - Minimum 14px font size
  • Match your brand - Use consistent colors across all panels
  • Use icons - Visual elements catch attention faster
  • Leave padding - Don't crowd the edges
  • Dark backgrounds - Match Twitch's dark mode

How to Add Panels to Twitch

  1. Go to your channel page on Twitch
  2. Click "Edit Panels" toggle (below video player)
  3. Click the + button to add a new panel
  4. Upload your image (320px wide, PNG or JPG)
  5. Add a link if it's clickable (social links, etc.)
  6. Add description text (optional)
  7. Click Submit

Twitch Panel Markdown

You can format panel descriptions using Markdown:

  • **bold** for bold text
  • *italic* for italic text
  • [link text](URL) for clickable links
  • - item for bullet lists

FAQ

What size should Twitch panels be?

320 pixels wide, and any height up to 600 pixels. Most streamers use 100-300px height for individual panels.

Can I use GIFs for Twitch panels?

Yes! Twitch supports animated GIFs for panels. Keep file size under 2.9MB.

How many panels can I have?

Twitch allows unlimited panels, but 6-10 is ideal. Too many becomes overwhelming.
