Frequently Asked Questions

What size should Twitch emotes be?

Twitch requires emotes in three sizes: 28x28 pixels (1x), 56x56 pixels (2x), and 112x112 pixels (4x). All emotes must be perfectly square.

What file format should Twitch emotes be?

Twitch emotes must be PNG format for best quality and transparency support. The file size must be under 1MB for the largest size.

Why won't my emote upload to Twitch?

Common issues: wrong dimensions (not square), file too large, or wrong format. Use this tool to check all requirements and resize as needed.
