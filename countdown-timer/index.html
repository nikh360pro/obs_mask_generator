Frequently Asked Questions

How do I add this timer to OBS?

Download the HTML file, then in OBS add a "Browser Source". Set the Local File option and select the downloaded HTML file. Set width to 800 and height to 200.

Can I use a transparent background?

Yes! Select "Transparent" background in the settings. In OBS Browser Source, the background will be transparent so the timer floats over your content.
